# Gerador de Links para Afiliados

## Sobre o Projeto

Aplicação web desenvolvida para automatizar a criação de ofertas para afiliados da Amazon.

O sistema recebe um link de produto, extrai automaticamente informações relevantes e gera uma mensagem pronta para divulgação.

## Problema Resolvido

O processo manual de criação de ofertas exige copiar informações do produto e montar mensagens repetitivamente.

O sistema automatiza esse fluxo e reduz significativamente o tempo necessário.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- Node.js
- Express
- Axios
- Cheerio

## Funcionalidades

- Extração de nome do produto
- Extração de preço
- Extração de imagem
- Geração automática de mensagem
- Processamento de múltiplos links
